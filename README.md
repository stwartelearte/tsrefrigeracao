# TS Refrigeração & Climatização

Landing page institucional e responsiva para apresentação dos serviços da TS Refrigeração & Climatização e conversão de solicitações pelo WhatsApp.

## Tecnologias

- React e TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Desenvolvimento

```bash
npm install
npm run dev
```

## Validação

```bash
npm run lint
npm run build
```

O deploy para o GitHub Pages é executado automaticamente a cada atualização da branch `main`.
